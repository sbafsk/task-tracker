# Check Assignment Progress

Check current assignment progress and completion status.

## What This Does

Reviews `docs/status/progress.yaml` and shows:
- Completed features
- Remaining tasks
- Test coverage status
- Assignment checklist completion

## When to Use

- Before working on new features
- After completing a feature
- To track overall progress
- Before final submission

## Output Format

Shows:
1. Overall completion percentage
2. Completed features (✓)
3. In-progress features (→)
4. Not started features (☐)
5. Test coverage summary
6. Next recommended task

## Quick Check

Current priority from `docs/status/progress.yaml`:
- Check high-priority tasks
- Identify blockers
- Review testing status
